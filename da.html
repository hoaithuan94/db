Lộ trình tự học Data Analyst cho newbie

1. Kiến thức về Toán và Thống kê:

  • Xác suất và thống kê
  • Đại số tuyến tính
  • Giải tích và toán rời rạc

    • 2. Kiến thức về Excel từ cơ bản đến nâng cao

  • Editing text and formulas
  • Excel functions and lists
  • Worksheets and Pivot Tables
  • Formatting data and data validation
  • Charts and templates
  • Lookup, Macros and VBA

    • 3. Ngôn ngữ lập trình Python

  • Syntax and Basics
  • Data Structures and Algorithms
  • Pandas
  • Numpy
  • SciPy
  • Matplotlib

    • 4. Ngôn ngữ truy vấn SQL và Database

  • DBMS, Normalisation and ERD
  • SQL Syntax, data types, variables, Select, Where, And, Or, Not
  • Insert, Update, Delete, Min, Max, Count, Average, Sum
  • Like, In, Between, Top, Group By, Order By, having, exists, any, all, case
  • Inner Join, Outer Join, Left Join, Right Join, Full Join, Self Join
  • Views, Triggers, Functions, PL/SQL, Injection, Hosting

    • 5. Công cụ trực quan hóa dữ liệu Power BI/ Tableau

  • Querying & Transforming Data
  • Data Modelling
  • Calculations and Formula
  • Reports & Visualisations
  • Dashboards

    • 6. Quy trình chuẩn bị dữ liệu (Data preparation) và validation

  • Data collection
  • Data discovery and profiling
  • Data cleansing
  • Data transformation
  • Data validation and publishing

    • 7. Exploratory Analysis và mô hình dữ liệu

  • Regressions
  • Classification
  • Clustering

    • 8. Các thư viện Machine Learning

  • PyTorch
  • Scikit - Learn
  • TensorFlow

    • 9. Kỹ năng kể chuyện bằng dữ liệu - Data Storytelling

  • Data storytelling là quá trình xây dựng và xử lý dữ liệu, trích xuất giá trị dữ liệu và truyền đạt giá trị đó đến những đối tượng mong muốn. Đây là cách thức hiệu quả để truyền đạt insights, ý tưởng theo cách thu hút nhất và giúp doanh nghiệp đưa ra quyết định dễ dàng hơn.
