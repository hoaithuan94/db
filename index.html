Summary:

Môn học giới thiệu các khái niệm cơ bản về cơ sở dữ liệu, hệ quản trị cơ sở dữ liệu và các mô hình dữ liệu. Tìm hiểu chi tiết về mô hình thực thể liên kết, mô hình dữ liệu quan hệ và khái niệm về ràng buộc dữ liệu, cụ thể là lý thuyết phụ thuộc hàm và chuẩn hóa dữ liệu; Trong học phần này sinh viên cũng được giới thiệu về hệ quản trị cơ sở dữ liệu MySQL được sử dụng để định nghĩa, thao tác và truy vấn dữ liệu, đồng thời làm quen với lập trình cơ sở dữ liệu thông qua lập trình T-SQL với công cụ trực quan MySQL WorkBench.

Đề cương chi tiết môn học: DC15113009.

Môn học trước: Không có

Môn học tiếp theo: Hệ quản trị cơ sở dữ liệu, Chuyên đề Oracle, Phân tích dữ liệu, Học máy và Khai phá dữ liệu v.v...

Tài liệu giảng dạy chính: Giáo Trình Cơ sở dữ liệu.

Phần mềm thực hành: MySQL Community Server và MySQL Workbench (xem hướng dẫn cài đặt).
